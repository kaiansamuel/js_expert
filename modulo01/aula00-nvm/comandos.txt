nvm install 20  -> instala a versão 18
nvm --lts -> instala a versão long-term support (versão recomendada)
nvm list
nvm use 18 -> use a 18 que ja está instalada na minha máquina
nvm alias default 18 -> defina a versão padrão do Node.js para ser a 18
criar .nvmrc -  com a versão desejada e nvm use para ativar aquela versão
node -v > .nvmrc -> salva a versão apropriada para aquele projeto
